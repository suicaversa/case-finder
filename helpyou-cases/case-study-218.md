# HELP YOU 活用事例: マニュアル作成・入力作業代行

## 基本情報
- **公開日**: 2019年8月29日
- **カテゴリ**: 経理
- **タグ**: 活用事例、資料作成、業務効率化

## タイトル
業務内容の見える化をサポート！マニュアル作成・入力作業代行は「HELP YOU」にお任せ！

## クライアント情報
- **業種**: 環境関連事業
- **課題**: 新規業務立ち上げにおける従業員レクチャーと不備対応の効率化

## 依頼内容
新規採用業務向けに開発されたアプリケーションの入力作業代行を依頼。マニュアル作成から実行まで包括的なサポートが必要でした。

## HELP YOUの提案フロー
1. 業務レクチャー
2. マニュアル作成
3. マニュアル提示
4. 入力作業実行
5. 完了報告

## 納品プロセス
- **第一段階**: マニュアル作成完了、クライアント確認実施
- **第二段階**: 入力作業完了、最終確認

## 工数削減成果

| 項目 | 詳細 |
|------|------|
| **HELP YOU担当工数** | 11時間 |
| クライアント工数 | 45分 |
| **総削減工数** | 11時間15分 |
| **削減率** | 依頼前12時間→確認作業45分のみ |

### 内訳
- マニュアル作成: 1時間
- 入力作業: 8時間
- チェック: 2時間

## 得られたメリット
- 従業員レクチャー時間削減
- 入力作業工数削減
- マニュアル化による内製化スムーズ化
- 業務の標準化・簡素化

## クライアント評価
"従業員レクチャーや不備対応の想定できない工数が削減され、スケジュール通りに事が運ぶことが増えました。"
