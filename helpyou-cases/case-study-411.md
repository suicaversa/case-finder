# HELP YOU 活用事例: カスタマーサポート業務効率化

## タイトル
「ここまでしてくれると思わなかった！期待以上のサポートで事務作業をカット」

## クライアント情報
- **業種**: 留学支援会社
- **依頼部門**: カスタマーサポート
- **公開日**: 2020年5月28日

## 課題
- お客様とのメールやり取りで時間を消費
- 明確な作業マニュアルが存在せず、感覚的に業務を実施
- 事務作業が蓄積し、プラスアルファの業務に着手できない

## 解決策
HELP YOUが提供したサポート内容：
1. 作業フローとマニュアルの作成
2. 使用アカウントの準備
3. メールテンプレートの作成
4. プログラム申し込みから請求までの事務業務代行

## 成果・メリット
クライアントの評価：「期待以上のサポートで、事務作業に取られる時間がほ"ぼなくなりました」

達成効果：
- **削減工数**: 4時間
- 来年計画立案の時間確保
- 業務マニュアル化の実現
- 複数人での業務共有基盤の構築

## 作業時間配分
- **HELP YOU**: 4時間30分
- **クライアント**: 1時間
